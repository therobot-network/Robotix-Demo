Request a New Distribution List or Membership Change

Summary

Distribution lists (DLs) in Microsoft 365 let you email a group of people using a single address. This article explains how to request a brand-new distribution list, add or remove members from an existing one, or change the list's properties (display name, owner, reply-to address).

Before You Begin

  • Only managers and IT-designated list owners can request changes to existing distribution lists on behalf of their teams.
  • If you are an individual employee requesting to be added or removed from a list you belong to, you can self-serve this in Outlook Web App for lists marked "Members can subscribe/unsubscribe." For locked lists, your manager must submit the request.
  • For new DL requests, have the following ready:
  • Desired email address (e.g., project-phoenix@summitit.com)
  • Display name (e.g., "Project Phoenix Team")
  • Initial member list (names or email addresses)
  • Owner name (who manages membership going forward)
  • Whether the list should accept email from outside the company (external senders)

Steps

Create a New Distribution List

  1. Go to https://help.summitit.com and navigate to Service Catalog → Accounts & Access → Distribution List — New Request.
  2. Fill in the New Distribution List form: - Requested Email Address: Must end in @summitit.com. IT checks availability and may suggest an alternative if the address is taken. - Display Name: How the group appears in the Global Address List. - Description: Briefly describe the group's purpose (visible in Outlook when someone searches for the list). - List Owner: The person responsible for managing membership. This can be yourself or a delegate. - Initial Members: Paste a comma-separated list of @summitit.com email addresses or employee names. - Accept External Email: Toggle on if the list needs to receive messages from non-Summit addresses. (This is off by default for security reasons.) - Moderated: Toggle on if you want messages to be approved by the owner before delivery to members. Useful for announcement lists.
  3. Click Submit. New lists are typically created within 1 business day. You receive a confirmation email when the list is live.

Add or Remove Members from an Existing List

Self-Service (for open lists)

  1. Open https://outlook.office365.com and sign in.
  2. Click the People icon in the left nav (or go to https://outlook.office365.com/people).
  3. In the search bar, type the distribution list name and click on it.
  4. Click Join (to add yourself) or Leave (to remove yourself) if the option is available. - If these buttons do not appear, the list is owner-managed — use the ticket method below.

Owner/Manager Request via Ticket

  1. Go to https://help.summitit.com and select Service Catalog → Accounts & Access → Distribution List — Membership Change.
  2. Fill in: - Distribution List Address (e.g., project-phoenix@summitit.com) - Action: Add Members / Remove Members / Both - Members to Add: Name or email of each person to add. - Members to Remove: Name or email of each person to remove. - Effective Date: Date you need the change active (defaults to ASAP).
  3. Submit. Changes are made within 1 business day.

Change Distribution List Properties

To change the display name, owner, email address, external-sender setting, or moderation, submit a Distribution List — Property Change ticket:

  1. Go to Service Catalog → Accounts & Access → Distribution List — Property Change.
  2. Enter the current list email address and describe what you want changed.
  3. Submit. IT processes property changes within 1–2 business days.

Troubleshooting

Symptom Likely Cause Fix
Desired email address is already taken Another DL or mailbox uses that address IT will suggest alternatives when processing your ticket
Emails sent to the new list bounce back List is not yet fully propagated in Exchange (can take up to 4 hours) Wait and retry; contact IT if still bouncing after 4 hours
Cannot see Join/Leave buttons in Outlook People List is owner-managed Submit a membership change ticket
External senders get a non-delivery report External email not enabled for this list Submit a Property Change ticket to enable external senders
List shows in Outlook search but is empty Members were not yet added or provisioning is in progress Check ticket status; allow 1 business day for member addition
