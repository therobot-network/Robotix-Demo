Connect to Summit-Guest Wi-Fi (Visitors)

Summary

Summit-Guest is Summit IT's visitor wireless network. It provides internet-only access through a captive portal and does not require Okta credentials or device enrollment. A daily access code is available from reception. This network is intended for guests, contractors, and personal devices. Corporate resources and internal applications are not accessible from Summit-Guest.

Before You Begin

  • Obtain the daily guest code from the front desk receptionist or your Summit IT host before attempting to connect.
  • The code changes every day at midnight MT and is different at each office location.
  • Summit-Guest is internet-only. If you need access to internal Summit IT applications, your corporate device must connect to Summit-Corp instead (see KB-0300).
  • No special software or certificates are required.

Steps

Windows 11

  1. Click the Wi-Fi icon in the taskbar (bottom-right).
  2. Select Summit-Guest from the list of available networks.
  3. Click Connect.
  4. Open a web browser (Edge, Chrome, etc.). You will be automatically redirected to the guest captive portal at guest.summitit.com. - If the portal does not appear automatically, navigate to http://guest.summitit.com manually.
  5. On the portal page, enter the daily guest code provided by reception.
  6. Accept the Terms of Use and click Get Online.
  7. You will see a confirmation page. You now have internet access.

macOS (Sonoma / Sequoia)

  1. Click the Wi-Fi menu in the menu bar and select Summit-Guest.
  2. macOS will typically open the captive portal sign-in window automatically. If it does not, open Safari and navigate to http://guest.summitit.com.
  3. Enter the daily guest code from reception.
  4. Accept the Terms of Use and click Get Online.
  5. You can now browse the internet.

iOS (iPhone / iPad)

  1. Open Settings > Wi-Fi and tap Summit-Guest.
  2. iOS will automatically open the captive portal sign-in screen. - If it does not open, tap the blue i next to Summit-Guest and tap Sign in to Network, or open Safari and go to http://guest.summitit.com.
  3. Enter the daily guest code and accept the Terms of Use.
  4. Tap Continue or Get Online.

Android

  1. Open Settings > Network & internet > Wi-Fi and tap Summit-Guest.
  2. Android typically shows a "Sign in to Wi-Fi network" notification — tap it. - If the notification does not appear, open Chrome and navigate to http://guest.summitit.com.
  3. Enter the daily guest code and accept the Terms of Use.
  4. Tap Get Online.

Other Devices (Linux, smart TVs, conference room equipment)

  1. Connect to the Summit-Guest SSID.
  2. Open a browser and navigate to http://guest.summitit.com.
  3. Enter the daily guest code and accept the Terms of Use.

Troubleshooting

Symptom Likely Cause Fix
Summit-Guest not visible in the Wi-Fi list Out of range or AP issue Move closer to the access point; ask reception if the guest network is active
Portal page does not load automatically Browser DNS quirk Manually navigate to http://guest.summitit.com in any browser
"Invalid code" on the portal Wrong or expired code Get the current day's code from reception; codes reset at midnight MT
Connected but no internet after portal Session not fully established Disconnect from Summit-Guest, reconnect, and repeat the portal sign-in
Portal says session expired Idle timeout (2 hours) Re-open any browser — you will be redirected to the portal to re-enter the code
Need access to internal apps Summit-Guest is internet-only Use your corporate device on Summit-Corp (KB-0300) or GlobalProtect VPN (KB-0302/KB-0303)
