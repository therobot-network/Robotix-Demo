Keep Your Computer Updated (Windows Update, Intune, Jamf)

Summary

Keeping your computer updated is one of the most important things you can do to protect Summit IT's systems. Security patches close vulnerabilities that attackers exploit; app updates fix bugs and add features. On Summit IT-managed devices, most updates are delivered automatically through Microsoft Intune (Windows) and Jamf Pro (macOS). This article explains how the update process works, how to check your current patch status, and how to manually trigger updates when needed.

Before You Begin

  • You do not need administrator rights to receive or install updates pushed by Intune or Jamf — they install automatically with elevated privileges.
  • Some updates require a device restart. Save all open work before initiating any update process.
  • Critical security patches (rated by Microsoft or Apple as "Critical") are enforced with a mandatory restart deadline. You will see countdown notifications before the forced restart occurs — do not ignore them.
  • If your device has not checked in with Intune or Jamf for more than 7 days (e.g., it was powered off), connect to GlobalProtect VPN and leave your device on and unlocked for 30–60 minutes to allow policies to sync.

Steps

Windows — Check Update Status in Company Portal

  1. Open Company Portal from the Start menu.
  2. Click Devices in the left navigation pane and select your device.
  3. Review the compliance status. If it shows Not compliant, there are outstanding updates or policy requirements. Click Check Status to refresh.
  4. To manually trigger a policy sync, click Sync on the device detail page. This tells Intune to push any pending updates immediately.

Windows — Check and Install Windows Updates Manually

  1. Click the Start menu and go to Settings (gear icon) > Windows Update.
  2. Click Check for updates. Windows will contact Microsoft's update servers and display available updates.
  3. If updates are found, click Download & install. Do not interrupt the download.
  4. When prompted, click Restart now or schedule a restart for a convenient time (use Schedule the restart to pick a time outside working hours).
  5. After the restart, return to Settings > Windows Update and run Check for updates again to confirm all updates were applied successfully.

Windows — View Installed Updates

  1. Go to Settings > Windows Update > Update history.
  2. Expand the Quality Updates and Security Updates sections to see recently installed patches, including their KB numbers and install dates.

macOS — Check Update Status via Self Service

  1. Open Self Service from your Applications folder or Dock.
  2. Navigate to the Updates category. Any pending IT-managed updates (e.g., CrowdStrike Falcon, Zoom, Slack) will appear here with an Update or Install button.
  3. Click Update for each item shown and follow the on-screen prompts.

macOS — Install macOS System Updates

  1. Click the Apple menu () > System Settings.
  2. Click General > Software Update.
  3. If updates are available, click Update Now or Upgrade Now (for a major macOS version upgrade — check with IT before upgrading to a new major version such as macOS Sequoia 15).
  4. Enter your macOS login password when prompted. The system will download and install updates, then restart.
  5. After the restart, return to System Settings > General > Software Update and confirm "Your Mac is up to date."

macOS — Trigger a Jamf Policy Check-In

If you suspect Jamf has pending policies waiting to deploy: 1. Open Self Service > Support section (if available) and click Sync Device Policies, or 2. Ask the Service Desk to push a remote policy sync to your device — this is done in under 2 minutes from the Jamf console.

Troubleshooting

Windows Update is stuck downloading at a certain percentage. Cancel the download by clicking Pause updates, wait 5 minutes, then click Resume updates. If it stalls again, restart your computer and try once more. If the issue persists after two attempts, contact the Service Desk with the KB number of the failing update.

I see a "Feature update to Windows 11, version 24H2" prompt — should I install it? Major feature updates are tested and approved by IT before being broadly deployed. If you see this prompt and have not received communication from IT about rolling out this update, do not install it manually — let Intune deploy it when the approved version is ready. Contact the Service Desk if you are unsure.

macOS Software Update says "An error occurred while checking for updates." Ensure you are connected to the internet or GlobalProtect VPN. If the error persists, go to Apple menu > Restart and try again after the restart. This error can also occur when Apple's update servers are under high load — try again in an hour.

Company Portal shows my device is "Not compliant" even after installing all updates. Click Sync in Company Portal and wait 15 minutes for the compliance status to refresh. If it still shows non-compliant, contact the Service Desk — there may be a configuration policy issue rather than a missing update.

I keep getting restart prompts but cannot restart right now because I am in a critical project. Use Schedule the restart in Windows Update to defer the restart to after business hours (e.g., 11:00 PM tonight). On macOS, click Remind Me Later but do not defer more than 24 hours for security patches. Mandatory deadlines cannot be extended by end users — contact the Service Desk if you need an IT-approved deferral.
