Create and Update Your Email Signature

Summary

A consistent email signature reflects professionally on Summit IT Services. This article shows you how to create or update your email signature in Outlook on Windows, macOS, and Outlook on the Web. Signatures set in the desktop app and the web app are managed separately — follow the steps for each client you use.

Before You Begin

  • Use the approved Summit IT signature format shown below. Do not modify fonts, colors, or add personal images/quotes without approval.
  • Your signature should include: full name, job title, department, phone number (if applicable), and the Summit IT Services logo (available on SharePoint: https://summitit.sharepoint.com/sites/brand).
  • Signatures configured in Outlook desktop do not automatically sync to Outlook on the Web — set it up in both places.

Approved Signature Format:

[First Name Last Name]
[Job Title] | [Department]
Summit IT Services
[Direct Phone] (optional)
summitit.com

Steps

Outlook on Windows

  1. Open Outlook and click File > Options.
  2. Select Mail in the left pane, then click Signatures… under the "Compose messages" section.
  3. In the Signatures and Stationery dialog, click New to create a signature or select an existing one to edit it.
  4. Give the signature a name (e.g., "Summit Standard") and click OK.
  5. In the text editor area, type your signature using the approved format above. Use the formatting toolbar to set the font to Calibri, 11pt, black.
  6. To add the Summit IT logo: place your cursor where you want the logo, click the Image icon in the toolbar, and browse to the logo file you downloaded from SharePoint.
  7. Under Choose default signature, set: - New messages: Summit Standard - Replies/forwards: Summit Standard (or "none" if you prefer no signature on replies)
  8. Click OK to save.

Outlook on macOS

  1. Open Outlook and go to Outlook > Preferences > Signatures.
  2. Click the + button to add a new signature.
  3. Double-click the signature name (e.g., "Untitled") and rename it to "Summit Standard."
  4. In the editor on the right, type or paste your approved signature. Format text using the toolbar — set font to Calibri, 11pt.
  5. To insert the Summit IT logo, drag the image file from Finder into the signature editor.
  6. Close the Signatures window. Outlook saves automatically.
  7. To set it as your default: open a new email, click Signatures in the message toolbar, and choose Summit Standard from the dropdown. To make it permanent for all new messages, use Outlook > Preferences > Signatures and assign it from the "Default signatures" section at the bottom.

Outlook on the Web (https://outlook.office.com)

  1. Sign in at https://outlook.office.com using your Okta credentials.
  2. Click the Settings gear icon in the top-right corner.
  3. In the Settings panel, type "signature" in the search box and select Email signature.
  4. In the signature text editor, type your approved signature. Use the formatting toolbar to set Calibri 11pt.
  5. To insert the logo, click the Insert image icon (picture frame) in the toolbar and upload the Summit IT logo from your computer or OneDrive.
  6. Check Automatically include my signature on new messages I compose and optionally Automatically include my signature on messages I forward or reply to.
  7. Click Save.

Troubleshooting

Symptom Action
Signature not appearing on new emails Confirm the signature is assigned under "New messages" in Signature settings.
Logo appears as a broken image to recipients Embed the image rather than linking it — use Insert > Picture (from file) not a URL.
Signature looks different on recipient's end Avoid using tables or custom HTML for complex layouts; keep formatting simple.
Can't find the Signatures option on macOS Make sure you are running the Microsoft 365 version of Outlook, not the legacy Outlook 2019 — open Help > About Outlook to verify.
Signature appears in OWA but not desktop Desktop and web signatures are separate; configure both using the steps above.
