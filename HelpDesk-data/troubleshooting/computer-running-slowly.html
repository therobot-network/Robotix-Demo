My Computer Is Running Slowly

Use this guide to diagnose and improve performance on a Summit IT-managed laptop (Dell Latitude 7440 or MacBook Pro 14") that is running slower than expected.

Start Here

Before diving in, characterize the slowness:

  • Is everything slow (startup, opening apps, typing) or only specific tasks (a particular app, video calls, large files)?
  • When did it start? Gradually over time, after a specific update, or suddenly today?
  • How old is the laptop? Check the asset tag — if it is approaching 4–5 years of service, hardware upgrades may be warranted.
  • Are system updates pending? Check Windows Update or macOS Software Update.

Run a restart first if you have not done so today — many performance issues are resolved by a simple restart.

Step-by-Step Diagnostic Flow

1. Have you restarted your laptop today?

If no → Restart now using Start > Power > Restart (Windows) or Apple menu > Restart (Mac). Do not simply close the lid (sleep mode) — a full restart clears memory, releases handles, and applies pending updates.

Did restarting improve performance significantly?

If yes → You likely had a memory leak or accumulated background processes. Make a habit of restarting at least once per week. Monitor for recurrence.

If no → Proceed to Step 2.

2. Are system updates pending?

Pending updates (including driver updates) can cause background processing that slows the system significantly. Apply all updates before further troubleshooting.

Windows: Go to Settings > Windows Update and click Check for updates. Install all available updates, then restart if prompted. Also check Company Portal for any pending Intune-managed software updates.

Mac: Go to System Settings > General > Software Update. Install all available updates. Also check Self Service for any pending Jamf-managed deployments.

Did installing updates and restarting resolve the slowness?

If yes → A background update process was consuming resources. No further action needed.

If no → Proceed to Step 3.

3. Check CPU and memory usage.

Windows: Press Ctrl + Shift + Esc to open Task Manager. Click the Performance tab to see overall CPU and memory usage. Then click Processes to see which applications or processes are at the top.

Mac: Open Activity Monitor (Applications > Utilities > Activity Monitor). Check the CPU and Memory tabs. Sort by % CPU or Memory to identify the top consumers.

Is a single process consuming more than 50% CPU or nearly all available RAM?

If yes, a known application (Outlook, Teams, browser) → Proceed to Step 4.

If yes, an unfamiliar process name → This could indicate a security issue. Note the process name and contact the Service Desk — do not attempt to end the process yourself if you do not recognize it.

If no (CPU and memory look normal) → Proceed to Step 5.

4. Is a specific high-CPU application causing the slowness?

  • Browser (Chrome/Edge): Too many open tabs are the most common cause of memory exhaustion. Close tabs you are not actively using, or use the browser's built-in "put tabs to sleep" feature. Consider installing uBlock Origin (approved extension — available in Company Portal) to reduce ad-related CPU load on web pages.
  • Microsoft Teams: Teams is known to be memory-intensive. Ensure it is up to date. Close extra Teams windows and disable hardware acceleration if on an older docking station (Teams settings > General > uncheck "Disable GPU hardware acceleration").
  • OneDrive sync: If OneDrive is actively syncing a large batch of files, it can spike CPU and disk I/O. Check the OneDrive icon in the taskbar for sync activity. Wait for it to complete.

Did addressing the high-CPU application improve performance?

If yes → Monitor and manage resource-heavy apps proactively.

If no → Proceed to Step 5.

5. Check available disk space.

Low disk space severely degrades performance, especially on the Windows system drive.

Windows: Open File Explorer, right-click Local Disk (C:), and select Properties to see free space. You should have at least 10 GB free. If less: empty the Recycle Bin, run Disk Cleanup (search in Start menu), and move large personal files to OneDrive (1 TB available).

Mac: Go to Apple menu > About This Mac > Storage. You should have at least 10 GB free. Remove large files or move them to OneDrive. Use the Manage button to see storage optimization recommendations.

Did freeing disk space improve performance?

If yes → Establish a habit of storing files in OneDrive rather than locally.

If no → Proceed to Step 6.

6. Check whether CrowdStrike or Intune/Jamf is running a scan or policy update.

CrowdStrike Falcon, Intune, and Jamf occasionally run intensive background tasks (full disk scans, software deployments, policy re-applications) that can temporarily slow the system. These are typically scheduled during off-hours but may run during business hours after a delay.

Windows: Open Task Manager > Processes and look for csfalconservice.exe, intunemanagementextension.exe, or msiexec.exe with high CPU/disk.

Mac: In Activity Monitor, look for falcond, jamfagent, or softwareupdated at the top of the list.

If one of these is at the top → Wait 15–30 minutes for the background task to complete. Do not force-quit these processes — they are security-critical. If the high usage lasts more than 2 hours, submit a ticket.

If none of these are responsible → Proceed to Step 7.

7. Run Windows built-in diagnostics (Windows only).

Search the Start menu for "Performance Monitor" or run perfmon /report in the Run dialog (Win + R). This generates a system diagnostic report. Also check Reliability Monitor (search Start menu) for a timeline of crashes, warnings, and errors that correlate with when the slowness started.

Did the report identify a specific cause? Note the finding and include it in your support ticket.

Mac: In Console (Applications > Utilities), look for repeated kernel panics, disk I/O errors, or thermal throttling messages in the system log.

8. Is the laptop very hot to the touch, or is the fan running at full speed continuously?

Thermal throttling occurs when the CPU reduces its speed to prevent damage from overheating.

If yes → - Ensure the laptop vents are not obstructed (do not use on soft surfaces like beds or sofas). - Clean dust from the vents using compressed air if accessible. - Dell Latitude: use on a flat, hard surface or a cooling pad. - If overheating continues, submit a ticket — the thermal paste may need reapplication or the cooling system may be failing.

If no → Proceed to Step 9.

9. How old is this laptop, and has it ever had a hardware upgrade?

Summit IT standard refresh cycle is 4 years. If your laptop is approaching or past this age:

  • Submit a Hardware Refresh Request via https://help.summitit.com under Hardware > Refresh Request.
  • Include your asset tag, purchase date (the Service Desk can look this up), and a description of the performance issues.

Quick Checklist

Before calling the Service Desk, confirm you have tried:

  • [ ] Restarted the laptop (full restart, not sleep/wake)
  • [ ] Applied all pending Windows / macOS updates and restarted
  • [ ] Checked Task Manager / Activity Monitor for high CPU or memory processes
  • [ ] Addressed known resource-heavy apps (Teams, browser tabs, OneDrive sync)
  • [ ] Verified at least 10 GB of free disk space on the system drive
  • [ ] Waited for CrowdStrike / Intune / Jamf background tasks to complete
  • [ ] Confirmed laptop is not overheating (hot vents, max fan speed)
  • [ ] Checked laptop age against the 4-year refresh cycle
