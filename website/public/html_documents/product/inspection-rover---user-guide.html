Installation Instructions

Step 1: Unpack and Inspect

Carefully remove all components and verify against the packing list. Check for any shipping damage. Document and report any discrepancies immediately to 1-800-555-ROBO.

Step 2: Base Installation

Mount the base unit using the provided M12 bolts. Torque to 45 Nm in a star pattern. Ensure the base is level within ±0.5°.

Step 3: Electrical Connections

Connect power supply (480V, 3-phase) through the main disconnect. Install emergency stop circuit using minimum 16AWG wire. Verify ground connection resistance is <0.1Ω.

Pre-Operation Safety Check

  • Verify emergency stop functionality
  • Check safety light curtain operation
  • Confirm work envelope clearance (minimum 1.5m)
  • Test all limit switches
