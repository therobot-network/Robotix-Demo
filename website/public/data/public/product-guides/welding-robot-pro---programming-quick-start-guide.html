  2. Home Position: Navigate to Setup > Calibration > Home Robot

4. Motion Commands

Basic motion command syntax:

Linear Movement (MoveL)

MoveL P1, v100, fine

Move linearly to position P1 at 100mm/s with fine positioning

Joint Movement (MoveJ)

MoveJ P2, v50, z10

Move in joint space to P2 at 50% speed with 10mm zone

Circular Movement (MoveC)

MoveC P3, P4, v200, fine

Circular movement through P3 to P4 at 200mm/s

Speed Parameter Description
v100 100mm/s TCP speed
v50 50% of maximum joint speed

5. Logic and Variables

Programming logic and variable handling:

Variable Types

  • num - Numeric values (integer or float)
  • bool - Boolean (TRUE/FALSE)
  • pos - Position data (X, Y, Z, Rx, Ry, Rz)
  • string - Text strings

Conditional Statements

IF DI[1] = TRUE THEN
  MoveL P1, v100, fine
ELSE
  MoveL P2, v100, fine
ENDIF

Loops

FOR i:=1 TO 10 DO
  MoveL PickPos[i], v100, fine
  GripperClose()
  MoveL PlacePos[i], v100, fine
  GripperOpen()
ENDFOR
