User Guide - CoBot Dual-Arm System

Congratulations on purchasing the CoBot Dual-Arm System from Robotix. This collaborative robot system represents the cutting edge in human-robot cooperation technology. Before proceeding with installation or operation, please read this guide thoroughly.

Installation Instructions

Step 1: Unpack and Inspect

Carefully remove all components and verify against the packing list. Inspect for any shipping damage. Report damages within 24 hours to support@robotix.com.

Step 2: Mounting Installation

Mount base plate to solid surface capable of supporting 250kg minimum load. Use M12 grade 8.8 bolts torqued to 85Nm.

Step 3: Electrical Connections

  • Connect main power cable (200-240VAC, 16A)
  • Wire emergency stop circuit
  • Connect safety barrier sensors
  • Verify ground connection (≤100Ω)
