Technical Specification: End-Effector Kit RBX-EEK-2024

Product Overview

The Robotix End-Effector Kit (RBX-EEK-2024) is a modular gripper system designed for seamless integration with collaborative and industrial robots. This versatile kit enables rapid deployment of custom end-effector solutions across multiple industries and applications.

Target Applications

  • Precision assembly operations
  • Material handling and sorting
  • Pick-and-place operations
  • Quality control inspection
  • Machine tending

Frame MaterialAircraft-grade aluminum 6061-T6
Gripper FingersHardened steel with quick-change interface
Weight0.85 kg (base unit)
Mounting PatternISO 9409-1-50-4-M6

Performance Specifications

Payload Capacity5 kg (maximum)
Grip Force10-120 N (adjustable)
Opening Width0-85 mm
Gripping Speed20-150 mm/s
Position Repeatability±0.02 mm

Electrical Specifications

  • Operating Voltage: 24V DC ±10%
