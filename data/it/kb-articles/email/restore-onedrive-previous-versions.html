Restore Previous Versions or Deleted Files in OneDrive

Summary

OneDrive for Business automatically saves version history for files and retains deleted files in the OneDrive Recycle Bin for 93 days. This article explains how to restore a previous version of a file and how to recover deleted files or folders from the OneDrive Recycle Bin.

Before You Begin

  • Version history is available for all file types stored in OneDrive for Business and SharePoint. It is not available for files stored only on your local device outside the OneDrive folder.
  • The OneDrive Recycle Bin retains deleted items for 93 days. After that, items are permanently removed.
  • SharePoint document libraries have their own separate Recycle Bin — see the SharePoint-specific steps below.
  • If you need to restore a large number of files after an accidental mass-deletion or ransomware event, the Service Desk can use the OneDrive Files Restore feature to roll back an entire OneDrive to a point in time (up to 30 days). Contact the Service Desk immediately in that scenario.

Steps

Restore a Previous Version of a File

Using Outlook on the Web / OneDrive web (Recommended):

  1. Go to https://robotix-my.sharepoint.com (your OneDrive) or navigate to the file in the OneDrive folder on your device and right-click.
  2. On the OneDrive website: select the file (click the circle checkbox next to it), then click ... (More options) in the toolbar and choose Version history. - From File Explorer (Windows): right-click the file > Show more options > Version History. - From Finder (macOS): right-click the file > Version History (requires OneDrive sync client).
  3. A panel opens on the right listing all saved versions with timestamps and the person who modified each version.
  4. To preview a version, click on the date/time link — it opens a read-only copy.
  5. To restore a version: click ... next to the version you want and choose Restore. This makes that version the current file without deleting newer versions (they are retained in history).
  6. To download a specific version as a copy: click ... > Download.

From within Microsoft 365 Apps (Word, Excel, PowerPoint):

  1. Open the file from OneDrive.
  2. Click the file name in the title bar at the top of the app.
  3. Select Version History from the dropdown menu.
  4. The version history panel opens on the right. Browse, preview, and restore as needed.

Recover Deleted Files from the OneDrive Recycle Bin

OneDrive on the Web:

  1. Go to https://robotix-my.sharepoint.com.
  2. In the left pane, click Recycle bin.
  3. Browse or search for the deleted file or folder.
  4. Select the item(s) by clicking the checkboxes.
  5. Click Restore at the top of the page. The item returns to its original location.
  6. If the original folder was also deleted, OneDrive recreates it automatically.

From File Explorer (Windows) or Finder (macOS):

The OneDrive Recycle Bin is separate from the Windows Recycle Bin or macOS Trash. Files deleted from your OneDrive folder go to the OneDrive Recycle Bin, not the local recycle bin.

  1. Right-click the OneDrive icon in the system tray (Windows) or menu bar (macOS).
  2. Select View online to open OneDrive on the web.
  3. Follow the web steps above to recover from the Recycle Bin.

Recover Deleted Files from a SharePoint Document Library Recycle Bin

If the file was in a SharePoint library (a Teams channel file, a shared site document library, etc.):

  1. Go to the SharePoint site (e.g., via the Teams channel > Files > Open in SharePoint).
  2. In the lower-left of the SharePoint site, click Recycle bin.
  3. Select the item and click Restore.
  4. If the Recycle Bin is empty but the file was deleted recently, it may be in the Second-Stage Recycle Bin (admin only) — contact the Service Desk.

Restore an Entire OneDrive to a Point in Time (Large-Scale Recovery)

If a large number of files were accidentally deleted or modified (e.g., mass overwrite or ransomware):

  1. Do not make any further changes to your OneDrive.
  2. Submit an urgent ticket at https://help.robotix.com marked P2 – High.
  3. The Service Desk will use OneDrive Files Restore to roll back your entire OneDrive to a specific date and time (available for up to 30 days).

Troubleshooting

Symptom Action
"Version History" option is not available The file must be saved in OneDrive or SharePoint. Files saved locally (e.g., Desktop, Documents outside OneDrive) do not have version history.
Restored version is not the content you expected Check the version list more carefully — the timestamps are in UTC by default. Adjust for your time zone when selecting.
Recycle Bin is empty but file was deleted recently If deleted from a shared library, check the SharePoint site Recycle Bin separately. The two Recycle Bins are independent.
Cannot restore — "You don't have permission" The file may be in a library owned by another team. Ask the site owner to restore it, or contact the Service Desk.
File restored but still missing in OneDrive folder on device Give OneDrive sync a few minutes to pull the restored file back down. Click the OneDrive icon and check sync status.
