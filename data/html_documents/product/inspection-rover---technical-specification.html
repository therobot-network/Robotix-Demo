Product Overview

The Robotix Inspection Rover (RBX-MR-2401) is an autonomous mobile robot designed for industrial inspection and monitoring applications. This versatile platform combines advanced sensor technology with robust mobility to deliver comprehensive facility inspection capabilities.

Target Applications

  • Manufacturing facility inspection
  • Infrastructure monitoring
  • Hazardous environment assessment
  • Quality control documentation
  • Security surveillance

Key Innovations

  • AI-powered obstacle avoidance system
  • Multi-sensor fusion technology
  • Real-time 3D mapping capabilities
  • Cloud-based inspection data management

Technical Specifications

Chassis/Frame

  • Material: Aircraft-grade aluminum alloy 6061-T6
  • Construction: Modular design with reinforced joints
  • Dimensions: 800mm x 600mm x 400mm (L x W x H)
  • Ground Clearance: 100mm
  • Weight: 45kg (base unit)

Mobility System

  • Drive Type: 4-wheel differential drive
  • Motors: 4x 250W brushless DC motors
  • Wheel Size: 200mm diameter, all-terrain rubber
  • Steering: Zero-radius turning capability

Control System

  • Controller: RBX-Control™ industrial-grade computer
  • Processor: Intel Core i7, 8th generation
  • Memory: 16GB RAM, 512GB SSD
  • Operating System: RBX-OS™ (Linux-based)
  • Connectivity: Wi-Fi 6, Bluetooth 5.0, 4G LTE

Sensor Package

  • Primary Camera: 4K resolution, 120° FOV
  • LiDAR: 360° scanning, 30m range
  • Thermal Camera: FLIR Lepton 3.5
  • Environmental Sensors: Temperature, humidity, gas detection
  • Positioning: GPS, IMU, wheel encoders
