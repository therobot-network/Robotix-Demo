Technical Specification: PalletMover Robot (RBX-PMR-2024)

1. Product Overview

The PalletMover Robot is an advanced autonomous mobile robot designed for efficient pallet transport and handling in warehouse and manufacturing environments. This next-generation mobile robot combines state-of-the-art navigation with robust payload capacity.

Key Applications:
  • Automated warehouse logistics
  • Manufacturing material transport
  • Cross-docking operations
  • Distribution centers
  • E-commerce fulfillment centers

2. Technical Specifications

Chassis/Frame:
  • Material: High-strength aluminum alloy frame with reinforced steel components
  • Construction: Modular design with protective panels
  • Dimensions: 1200mm x 900mm x 400mm (L x W x H)
  • Ground Clearance: 35mm
Drive System:
  • Motor Type: Brushless DC servo motors (4x)
  • Drive Configuration: Mecanum wheel system
  • Motor Power: 750W per motor
  • Steering: 360° omnidirectional movement
Control System:
  • Controller: RobotiX RC-750 Industrial Controller
  • Processor: Intel Core i7, 16GB RAM
  • Operating System: RobotiX OS 4.0
  • Communication: WiFi 6E, Bluetooth 5.2, Ethernet
  • Programming Interface: REST API, Python SDK
Sensors:
  • Navigation: LiDAR (270° field of view)
  • Safety: 3D Time-of-Flight cameras (x4)
  • Positioning: SLAM with magnetic tape guidance backup
  • Load Detection: Weight sensors with ±0.1kg accuracy
