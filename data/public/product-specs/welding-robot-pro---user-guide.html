Welding Robot Pro

User Guide & Setup Instructions

Introduction

Congratulations on your purchase of the Welding Robot Pro from Robotix. This advanced robotic welding system represents the culmination of 25 years of robotics innovation. Before operating this equipment, please read this manual thoroughly.

WARNING: Robotic welding systems involve high voltages, moving components, and heat sources that can cause serious injury or death if not operated properly. Only qualified personnel should install, program, and maintain this equipment.

This product complies with ISO 10218-1:2011, ANSI/RIA R15.06-2012, and carries CE marking certification.

Installation Instructions

Step 1: Unpack and Inspect

Carefully remove all components and verify against the packing list. Inspect for any shipping damage. Report any discrepancies to Robotix within 48 hours.

Step 2: Base Installation

Mount base plate to level concrete floor using M16 anchor bolts (supplied). Torque to 120 Nm. Maximum floor deviation: 0.5mm/m.

Step 3: Electrical Connections

Connect 3-phase power (380-480VAC, 50/60Hz) through supplied circuit breaker. Install emergency stop circuit using minimum 1.5mm² wire. Reference drawing E-2234 for terminal locations.

Pre-Operation Safety Check

  • Verify emergency stop functionality from all stations
  • Test safety light curtain system
  • Check all limit switches
  • Verify teach pendant dead-man switch
  • Confirm proper grounding (≤100Ω)
