Product Overview

The RBX-HSA-2024 HighSpeed Assembly Robot represents Robotix's next-generation solution for high-precision automated assembly operations. Designed specifically for electronics, automotive, and medical device manufacturing, this versatile system delivers industry-leading speed and precision while maintaining exceptional reliability.

Target Applications

  • PCB assembly and electronics manufacturing
  • Automotive components assembly
  • Medical device assembly
  • Consumer electronics production
  • Small parts handling and precision assembly

Key Innovations

  • Advanced vision-guided positioning system with 0.02mm accuracy
  • Proprietary force-feedback control system
  • AI-powered path optimization
  • Industry 4.0 ready with comprehensive IoT integration

Technical Specifications

Mechanical Configuration

Frame Construction:Aerospace-grade aluminum alloy
Robot Weight:280 kg
Reach:850mm
Maximum Payload:12 kg
Degrees of Freedom:6-axis articulated

Drive System

  • Brushless AC servo motors with absolute encoders
  • Harmonic drive reduction gears
  • Zero-backlash precision gearing

Control System

  • RBX-C2000 Advanced Controller
  • Real-time path planning at 2ms intervals
  • Ethernet/IP, Profinet, Modbus TCP connectivity
  • 15" touchscreen teach pendant

Sensor Package

  • Integrated 3D vision system with dual 5MP cameras
  • 6-axis force/torque sensor
  • Collision detection sensors

Performance Metrics

Maximum Speed:8 m/s
Acceleration:40 m/s²
Repeatability:±0.02mm
Position Accuracy:±0.05mm
Cycle Time:0.38s (standard pick-and-place)
Duty Cycle:100% at rated load

Operating Requirements

Power Supply:200-240VAC, 50/60Hz, 3-phase
Power Consumption:2.8 kW typical
Operating Temperature:5°C to 45°C
Protection Rating:IP65 standard (IP67 optional)
