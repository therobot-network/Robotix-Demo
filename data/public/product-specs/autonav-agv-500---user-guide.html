Installation Procedure

  1. Initial Setup
    • Unpack and inspect all components
    • Verify facility floor meets minimum flatness requirement (FF25)
    • Install charging station according to provided template
  2. Safety System Setup
    • Mount emergency stop buttons at specified locations
    • Configure safety laser scanner zones
    • Test all safety interlocks
  3. Software Configuration
    • Install AutoNav Controller Software v4.2 or higher
    • Configure network settings (Static IP recommended)
    • Upload facility map and define routes
