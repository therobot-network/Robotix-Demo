Installation Instructions

Step 1: Site Preparation

Ensure mounting surface can support 120kg minimum load. Required concrete thickness: 150mm minimum. Ambient temperature must be 5-40°C with 20-80% relative humidity.

Step 2: Base Installation

  1. Mark mounting holes using provided template
  2. Drill 12mm diameter holes, 100mm deep
  3. Install M10 anchor bolts (supplied)
  4. Torque to 40Nm using calibrated torque wrench

Step 3: Electrical Connections

Connect power and control cables according to provided wiring diagram. All connections must be performed by qualified electrician.

⚠️ DANGER: Ensure main power is locked out before making any electrical connections.

Pre-Operation Safety Checklist

  • Verify Emergency Stop functionality
  • Check safety barrier interlocks
  • Confirm teach pendant operation
  • Test limit switches
  • Verify all mounting bolts are torqued to specification

Maintenance Schedule

Daily Checks:

  • Visual inspection of cables and connections
  • Emergency stop function test
  • Check for unusual noise or vibration

Monthly Maintenance:

  • Lubricate axis joints (use only Robotix-approved lubricant RL-220)
  • Check and clean ventilation filters
  • Verify axis backlash measurements
  • Backup system configuration
