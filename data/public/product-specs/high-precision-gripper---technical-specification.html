Product Overview

The RBX-GRP-2024 High-Precision Gripper represents Robotix's latest advancement in robotic end-effector technology. Designed for high-precision assembly and handling applications, this gripper combines advanced force control with innovative sensor fusion for unprecedented accuracy and versatility.

Target Applications

  • Semiconductor manufacturing
  • Medical device assembly
  • Electronics manufacturing
  • Precision mechanics assembly
  • Laboratory automation

Key Innovations

  • Dual-mode operation (force/position control)
  • Integrated multi-modal sensing
  • Advanced haptic feedback system
  • AI-powered grip force optimization

Technical Specifications

Mechanical Properties

Frame MaterialAircraft-grade aluminum 7075-T6
Finger MaterialHardened stainless steel with replaceable TPE coating
Weight450g
Grip Force Range0.5N - 150N
Stroke0-100mm

Actuator System

  • Brushless DC servo motor with planetary gearhead
  • Integrated absolute encoder (19-bit resolution)
  • Closed-loop position/force control
  • Maximum speed: 250mm/s

Sensor Suite

  • 6-axis force/torque sensor
  • High-resolution tactile array (256 elements)
  • Dual optical proximity sensors
  • Temperature monitoring

Control System

  • 32-bit ARM Cortex-M7 microcontroller
  • EtherCAT real-time communication
  • Built-in collision detection
  • Programmable via RobotixStudio software
