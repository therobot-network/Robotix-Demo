Installation Instructions

Step 1: Unpack and Inspect

Carefully remove all components and verify against the packing list. Check for any shipping damage. Operating voltage: 24V DC ±10%.

Step 2: Base Installation

Mount the base unit using the provided M12 bolts. Torque to 45 Nm. Ensure the mounting surface can support 250kg minimum load.

Step 3: Electrical Connections

Connect power supply and safety circuits according to provided wiring diagram. Emergency stop must be wired to pins E1 and E2 on the safety controller.

Pre-Operation Safety Check

  • Verify Emergency Stop functionality at all stations
  • Test safety light curtain operation (response time < 100ms)
  • Check work envelope clearance (minimum 500mm from fixed objects)
  • Verify all sensors are properly calibrated

Maintenance Schedule

Daily Checks

  • Visual inspection of cables and connectors
  • Emergency stop function test
  • Clean proximity sensors with compressed air

Monthly Maintenance

  • Lubricate wheel bearings (use supplied RX-400 grease)
  • Check and tighten all mounting bolts
  • Calibrate position sensors

Troubleshooting Common Issues

Issue Solution
Error Code E-201 Check emergency stop circuit connections
Position Drift Recalibrate floor markers, check wheel encoders
