Product Overview

The RBX-HDR-2024 is a heavy-duty 6-axis articulated robot designed for demanding industrial applications requiring high payload capacity and precise manipulation. Built with advanced servo technology and robust construction, it delivers exceptional performance for manufacturing, assembly, and material handling operations.

Target Applications

  • Heavy component assembly
  • Automotive manufacturing
  • Machine tending
  • Palletizing/depalletizing
  • Metal fabrication

Mechanical Structure

Base MaterialCast iron with reinforced steel frame
Arm ConstructionAluminum alloy with carbon fiber components
Weight1,250 kg
Reach2,850 mm
Payload Capacity180 kg

Motion System

  • Servo Motors: Fanuc α-iS 50/3000 series
  • Encoders: 20-bit absolute encoders
  • Brakes: Dynamic electromagnetic on all axes
  • Joint Ranges:
    • J1: ±180°
    • J2: -60° to +90°
    • J3: -180° to +75°
    • J4: ±360°
    • J5: ±120°

Control System

  • Controller: RBX-C2000 Industrial Controller

  • Programming: RBX-Studio Software, Python API
