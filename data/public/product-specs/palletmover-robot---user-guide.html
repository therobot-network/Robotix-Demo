User Guide - PalletMover Robot

Introduction

Congratulations on purchasing the Robotix PalletMover Robot, a state-of-the-art automated material handling solution. This guide will help you safely install, operate, and maintain your system.

WARNING: Robotic systems involve moving machinery that can cause serious injury. Always follow safety protocols and keep personnel clear of the robot's operating envelope during operation.

This product complies with ISO 10218-1:2011, ANSI/RIA R15.06-2012, and carries CE marking for European markets.

Installation Instructions

Step 1: Site Preparation

Ensure installation area meets minimum space requirements: 3m x 3m work envelope, plus 1.5m safety perimeter. Floor must be level within ±2mm/m.

Step 2: Base Installation

Mount base unit using provided M12 anchor bolts. Torque to 45 Nm. Ensure base is level within 0.1°.

Step 3: Electrical Connections

Connect power supply (24V DC, 20A max). Install emergency stop circuit using provided safety-rated components.

Pre-Operation Safety Checklist

  • Verify emergency stop functionality
  • Test safety light curtains
  • Check all mounting bolts are torqued correctly
  • Verify sensor calibration
  • Test communication with control system

Maintenance Schedule

Daily Checks:

  • Visual inspection of cables and connectors
  • Clean sensor surfaces
  • Check emergency stop function

Monthly Maintenance:

  • Lubricate all specified points
  • Check belt tension
  • Verify all bolts are tight to spec
  • Test safety systems

Troubleshooting

Error Code E001: Communication Loss

  • Check network cable connections
  • Verify controller power supply
  • Restart control system
