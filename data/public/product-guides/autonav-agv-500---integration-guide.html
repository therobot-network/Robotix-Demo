5. Communication Protocols

Supported communication protocols:

Industrial Ethernet

  • Modbus TCP: Port 502, slave mode
  • EtherNet/IP: Scanner/Adapter modes
  • PROFINET: IO-Device mode, RT/IRT
  • OPC UA: Server implementation

Digital I/O

  • 24 configurable digital inputs
  • 16 configurable digital outputs
  • 24VDC, sourcing/sinking configurable
  • Response time: < 1ms

7. API Reference

REST API endpoints for external system integration:

Authentication

POST /api/v1/auth/login

Authenticate and receive JWT token

Robot Control

GET /api/v1/robot/status

Retrieve current robot status

POST /api/v1/robot/move

Execute movement command

POST /api/v1/robot/program/start

Start program execution

Data Monitoring

GET /api/v1/robot/position

Get current position (X, Y, Z, Rx, Ry, Rz)

GET /api/v1/robot/io

Read I/O status
